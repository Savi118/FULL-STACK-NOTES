Glossary of Web Terms

HTML
HyperText Markup Language — the standard language used to create web pages.
CSS
Cascading Style Sheets — used to style and layout web pages.
JavaScript
A programming language used to make web pages interactive and dynamic.
URL
Uniform Resource Locator — the address used to access resources on the internet.
HTTP
HyperText Transfer Protocol — the protocol used for transferring web content.
Responsive Design
A design approach that makes web pages look good on all devices (mobile, tablet, desktop).
